## Установка
Убедиться, что у вас установлены docker и docker-compose. 
<a name="install"></a>
<a name="installstart"></a>
```sh
npm install
```

```sh
npm run docker:stage
```

Игра запустится на https://localhost:5000

## Тестирование + Линтеры

## Дизайн
### [Дизайн в Фигме](https://www.figma.com/file/VQfJFBznJXtKE6i0qTMfLe/Layout?node-id=0%3A1)

## Установка
<a name="install"></a>
<a name="installstart"></a>
```sh
npm install
```

```sh
npm run start
```

Игра откроется автоматически на http://localhost:9000

## Тестирование
<a name="install"></a>
<a name="installstart"></a>
```sh
npm run test
```

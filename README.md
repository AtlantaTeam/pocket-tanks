# Игра Pocket Tanks

## Дизайн
Здесь будет ссылка на дизайн в Фигме

## Установка
<a name="install"></a>
<a name="installstart"></a>
```sh
npm install
```

```sh
npm run start
```

Затем идем на http://localhost:9000

## Тестирование
<a name="install"></a>

```sh
npm run test
```
